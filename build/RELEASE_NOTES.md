## MICO360 Doc Toolkit v5.6.0

**New: Word → Markdown.**

- **Word → Markdown** — bulk-convert Word documents (`.docx`, and `.doc`/`.odt`/`.rtf`
  via LibreOffice) to clean Markdown (`.md`). Keeps headings, **bold** / *italic*,
  bullet & numbered lists, and tables. Convert a whole folder at once.

Includes everything from v5.5.x: the design refresh (bold sidebar headings,
consolidated Organize PDF, Protect confirm/show-hide, Watermark position grid,
Settings tabs), **2.4× faster OCR** on large scans (identical output), and the
smaller install from the trimmed bundle.
